## Levantar servidor

Desde la raiz del proyecto para levantar, mySQL, phpMyAdmin y el servidor WebSocket
```
docker compose up --build
```

### Para levantar angular desde Pichon/frontend
```
ng serve -0
```

### Acceso a Phpmyadmin

- http://127.0.0.1:8080/
